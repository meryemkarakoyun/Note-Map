# 🗺️ Note Map
**Note Map** is a modern, responsive web app that lets you drop pin‑point notes on an interactive Leaflet map, view them with smooth fly‑to animations, and manage them through a stylish glass‑morphism sidebar. 
## 🚀 Features
- 📍 **Interactive map** with OpenStreetMap tiles (Leaflet)
- ✈️ **Fly‑to animation** when clicking a note’s airplane icon
- 🧾 **Collapsible sidebar** (glass‑morphism, dark overlay)
- 📝 **Add / delete notes** (title, date, status)
- 💾 **Persistent storage** in `localStorage`
- 📱 **Fully responsive** – works on desktop, tablet & mobile
- 🎨 **Modern UI** – gradients, micro‑animations, custom Google Font **Poppins**
